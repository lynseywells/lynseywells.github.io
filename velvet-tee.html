Velvet-Tee Custom T-Shirts

An online order form utilizing JavaScript.

Project Summary

This single web page is an online order form for a fictional tee-shirt printing company. JavaScript is used to enable or disable form fields depending on what the user inputs prior. This is done dynamically, as the user interacts with the field. For example, if a field required for another to be active is cleared, the associated field will be disabled. When a field is disabled or enabled, its style is also changed.

Upon form submission, JavaScript is also used to validate the data entered into the form. The user is alerted if there are errors and prevented from submitting the order. If the form is valid, the user is notified that the order went through and the form is reset. Event listeners were added to buttons so the user can calculate the orders subtotal before finalization or to reset the form.
