Bizmart Supply Co.

A retail application built in Java.

Project Summary

This project is a three-in-one program built with Java for a fictional business; it has a POS system, a business management tool, and an online shop. Different features are available depending on a users account type. The projects goal is to provide Bizmart Supply Co a user-friendly way to manage data and conduct business both online and in-person. When the user signs in, they are presented with a different interface depending on the accounts type: either customer, employee, or manager.

The customer view allows users to search for items, add them to a cart, and place an order. The employee view is a POS system to process transactions, and then create a receipt. The manager view allows managers to generate HTML sales reports, manage account data for all accounts, create discount codes, and manage product inventory.

This application relys on an SQLite database to keep track of inventory, user information, et cetera. The program manages an open database connection and inserts/updates rows based on user input. For example, placing an order subtracts the amount of a product ordered from the database, preventing other users from order more than what's currently in stock.
