Rat Care Website

An online guide for taking care of fancy rats.

Project Summary

This website is a detailed guide for taking care of fancy rats. I researched rat care and then used the information I learned to write all of the content on the site. After that I used CSS to style the page, including a media query to change the sites layout for smaller devices.

It consists of 4 pages with navigation between each page. Each page is marked up properly with semantic elements to ensure the website is easily accessible to screen readers. For example, headings are used to indicate importance rather than being used for styling, and sections are used to organize content and are labeled with a title attribute. I used the W3C validator to check that all elements were used properly.
